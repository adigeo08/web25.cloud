What is Web25.Cloud?

                        
Decentralized web platform built on peer-to-peer technology

                    


                    

                        

                            

                                Web25.Cloud is a decentralized web platform built on top of
                                PeerWeb. It extends peer-to-peer website
                                hosting with a local in-browser signing wallet,
                                signed torrent deployments, and browser-native distribution without a central server.
                            

                        

                    


                    

                        

                            

                                
💾

                                
Smart Caching

                                
Visited sites are cached for lightning-fast repeat loads.

                                
    
                                    
  • 🏃‍♂️ Instant Loading — Cached sites load immediately
    • 
                                    
  • 💽 IndexedDB Storage — Persistent across browser sessions
    • 
                                    
  • 🧹 Auto Cleanup — Old cache expires after 7 days
    • 
                                    
  • 🔒 Secure Storage — Only cached after DOMPurify sanitization
    • 
                                

                            


                            

                                
🛡️

                                
Security Features

                                
Multi-layer security with DOMPurify integration.

                                
    
                                    
  • 🧼 XSS Protection — All HTML sanitized with DOMPurify
    • 
                                    
  • 🚫 Malicious Code Removal — Dangerous tags and attributes filtered
    • 
                                    
  • 📦 Sandboxed Execution — Sites run in isolated iframe environment
    • 
                                    
  • 🔍 Content Validation — All resources validated before display
    • 
                                

                            

                        

                    


                    

                        

                            

                                
📋

                                
Requirements for Hosted Sites

                            

                            

                                
📄 Must contain an index.html file in the bundled artifact

                                
🔗 All resources should use relative paths

                                
⚡ Only static content (HTML, CSS, JS, images, fonts, etc.)

                                
🔒 Files are served in a sandboxed environment for security

                                
🧠 Deploy flow seeds from an in-memory bundle, not directly from local folder handles

                                
✅ On load, signature metadata is verified and mirrored into manifest.web25.json

                                
📱 Responsive design recommended for best experience

                                
🧹 Content is automatically sanitized for security
